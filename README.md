Transfer Labels is a mod that allows you to transfer items/fluids between blocks that are right next to each other.

### How it works:

*   You can place labels on blocks by sneak right clicking into blocks
*   You can interact with the label by holding a label and right-clicking the placed label
*   You can remove the placed label by left-clicking the label while holding a label

You can access labels through blocks using the label accessor

### Filtering:

When opening the label GUI you can filter with different filter types like regulating, exact count, mod and tag.

For the filter that have an amount, you can change how much while scrolling while holding Shift/Alt/Ctrl and any combination of those keys

### Acceleration:

You can accelerate the labels by adding the same item label in the amount or speed slots.
